# Heliophorus-androcles
A simple checksum calculator packaged in a Docker container.
Using the [coreutils](https://www.gnu.org/software/coreutils/manual/html_node/Summarizing-files.html) hashing tools, with [JC](https://kellyjonbrazil.github.io/jc/docs/parsers/hashsum) to provide JSON output.

This project is named after the [Heliophorus androcles](https://en.wikipedia.org/wiki/Heliophorus_androcles) butterfly.
